232 nips-2012-Multiplicative Forests for Continuous-Time Processes


Source: pdf

Author: Jeremy Weiss, Sriraam Natarajan, David Page

Abstract: Learning temporal dependencies between variables over continuous time is an important and challenging task. Continuous-time Bayesian networks effectively model such processes but are limited by the number of conditional intensity matrices, which grows exponentially in the number of parents per variable. We develop a partition-based representation using regression trees and forests whose parameter spaces grow linearly in the number of node splits. Using a multiplicative assumption we show how to update the forest likelihood in closed form, producing efÔ¨Åcient model updates. Our results show multiplicative forests can be learned from few temporal trajectories with large gains in performance and scalability.
