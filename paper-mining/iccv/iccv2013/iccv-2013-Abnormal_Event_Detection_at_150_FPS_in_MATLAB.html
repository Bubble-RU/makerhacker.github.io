34 iccv-2013-Abnormal Event Detection at 150 FPS in MATLAB


Source: pdf

Author: Cewu Lu, Jianping Shi, Jiaya Jia

Abstract: Speedy abnormal event detection meets the growing demand to process an enormous number of surveillance videos. Based on inherent redundancy of video structures, we propose an efficient sparse combination learning framework. It achieves decent performance in the detection phase without compromising result quality. The short running time is guaranteed because the new method effectively turns the original complicated problem to one in which only a few costless small-scale least square optimization steps are involved.
