61 acl-2010-Combining Data and Mathematical Models of Language Change


Source: pdf

Author: Morgan Sonderegger ; Partha Niyogi

Abstract: English noun/verb (N/V) pairs (contract, cement) have undergone complex patterns of change between 3 stress patterns for several centuries. We describe a longitudinal dataset of N/V pair pronunciations, leading to a set of properties to be accounted for by any computational model. We analyze the dynamics of 5 dynamical systems models of linguistic populations, each derived from a model of learning by individuals. We compare each modelâ€™s dynamics to a set of properties observed in the N/V data, and reason about how assumptions about individual learning affect population-level dynamics.
